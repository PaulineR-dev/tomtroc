# TomTroc

TomTroc est un site web permettant aux utilisateurs d’échanger leurs livres entre eux.  
Le projet est développé en PHP avec une architecture MVC.

---

## Fonctionnalités

- Inscription et connexion des utilisateurs  
- Page “Mon compte” avec modification du profil  
- Bibliothèque personnelle (ajout, édition, suppression de livres)  
- Statut de disponibilité des livres  
- Page “Nos livres à l’échange” avec recherche par titre  
- Page de détail d’un livre  
- Page de profil public d’un utilisateur  
- Messagerie simple (voir messages, répondre, envoyer)

---

## Architecture

Le projet utilise une structure MVC :
/controllers
/models
/views
/scripts
/config
index.php (routeur)


---

## Technologies

- PHP  
- HTML / CSS  
- MySQL
- JS

---

## Installation

1. Cloner le projet  
2. Importer la base de données  
3. Configurer la connexion dans `config/config.php`  
4. Lancer le projet via : http://localhost/tomtroc/index.php?action=home
